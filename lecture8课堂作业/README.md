##### 存储过程练习
存储过程、函数与触发器在真实业务系统中都适用于具体哪些场景，用自己的话说明一下原因。
存储过程的应用：当希望在不同的应用程序或平台上执行相同的函数，或者封装特定功能时。   
原因：一个存储过程是一个可编程的函数，它在数据库中创建并保存。它可以有SQL语句和一些特殊的控制结构组成。存储过程是非常有用的。数据库中的存储过程可以看做是对编程中面向对象方法的模拟。它允许控制数据的访问方式。
